Introduction





1. Scene View is used for the interactive design of scenes and 3d worlds
2. Game View serves as a preview of the game
3. Hierachy shows all game objects used in the scene
4. Inspector shows all components and public parameters of the currently selected game object
5. Project Browser serves to display and manage all digital content.
by dragging them into the hierarchy, they are inserted into the scene
6. Console serves to display error and information messages

The arrangement of the respective windows can be changed as desired.
The dropdown layout menu in the top right corner of the Unity window serves as a quick access for already prepared layouts and for saving the self-created layout.
