What is C?

C is a general-purpose computer programming language.

C was designed at Bell Laps in 1972/1973 an originally ran on the Unix-OS. During the years, C has become one of the most popular computer programming languages in the world.

It was designed to work realtively straightforward with different compilers and it is pretty handy on memory.

Nowadays you can program with C on every Operating System you wnat. The latest stable release was C18 in June 2018.

As you may have noticed, there are similarities between C and C# so if you learn the one the other one is much easier to start, because they share many different characteristics

What is Repl.it

Repl.it is a online compiler which we as developers of this website are also using regulary.

It is an american start-up which is supporting more than 50 different programming languages and you don't need to register at any time

So it is freely available to everyone and it is easy to use
