What is C#?

C# (pronounced: "see sharp") is a general-purpose, genric, component-oriented and object oreinted programming language. It has been developed by Microsoft since 2000. It was designed by Anders Hejlsberg.

C# is a simple programming language, which is easily acessible, it has strong similarities with its naming-partners C and C++, regarding programming structure.

The newest Version of C# is Version 8.0 which was released in Semptember 2019 together with the newest Visual Studion Version (VS19 16.5)

What is Visual Studio?

Microsoft Visual Studio is development environment, also developed by Microsoft, Visual Studio makes programming much easier and there's also an free Version of it Visual Studio Community which you can download left in the sidebar or here.

There are also the Proffesional and Enterprise Versions of Visual Studio which are free for 30 days before you have to buy the licence. This licence is about 1100$

When you start the installment of Visual Studio make sure you select "Universal Windows Platform development" and ".NET desktop development" the total Install Size should be around 15 to 20 Gigabytes
